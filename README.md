# NeosEnviSim

---
Tools for **building environment simulation** with Grasshopper.
Developed by **Huafei Huang**, since **2025**.
Based on **Rhino 8-Grasshopper 1.8**, with continuous expansion and improvement.
---

**NeosEnviSim** is a suite of building environmental performance analysis tools based on **Rhino-Grasshopper**.
It offers a wide range of functionalities, including：
**·Non-visual light efficacy**
**·Sunlight duration**
**·Solar radiation**
**·Bifacial photovoltaics**
**·Soil energy balance**
**·Outdoor thermal comfort**
**·View factor**
**·Sound capture**
**·Pedestrian simulation (evacuation time)**
**·Street view analysis**
**·Data visualization**
**·...**
